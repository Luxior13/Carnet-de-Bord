# Acceptation des chartes

## Objectif

Suivre quels membres ont lu et accepte les chartes ou documents officiels de la structure.

## Ce que la feature doit permettre

- Demander l'acceptation d'une charte.
- Lier une acceptation a une version precise du document.
- Voir qui a accepte.
- Voir qui n'a pas encore accepte.
- Relancer les membres en attente.
- Garder l'historique des acceptations.
- Bloquer certaines actions si une charte obligatoire n'est pas acceptee.

## Donnees a garder

- Document concerne.
- Version du document.
- Membre concerne.
- Date d'acceptation.
- Adresse IP si necessaire.
- Statut.
- Relances envoyees.

## Statuts possibles

- En attente.
- Accepte.
- Refuse.
- Expire.

## Permissions

- Voir les acceptations.
- Demander une acceptation.
- Relancer un membre.
- Voir l'historique.
- Marquer une charte comme obligatoire.

## A clarifier

- Si l'acceptation a une valeur officielle ou seulement interne.
- Si tous les membres doivent accepter toutes les chartes.
- Si une nouvelle version relance automatiquement une acceptation.
