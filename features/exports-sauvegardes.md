# Exports et sauvegardes

## Objectif

Permettre de sortir les donnees importantes du site prive et prevoir une logique de sauvegarde.

## Role dans le site

- Exporter la tresorerie.
- Exporter les membres ou adherents.
- Exporter les sponsors.
- Exporter les contrats ou documents.
- Exporter les archives.

## Organisation UX

Les exports globaux sont dans Systeme.
Les exports financiers restent dans le pole Tresorerie.
