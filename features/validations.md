# Validations

## Objectif

Prevoir une file de validation pour les actions sensibles du site prive.

## Role dans le site

- Valider une grosse depense.
- Valider un contrat.
- Valider une sanction.
- Valider un document officiel.
- Valider une archive ou suppression sensible.

## Organisation UX

Validations peut etre visible dans Systeme pour les validations globales et dans Tresorerie pour les validations financieres.
